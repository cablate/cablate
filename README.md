## What I Build

### AI Infrastructure

- [mcp-google-map](https://github.com/cablate/mcp-google-map) — Google Maps through MCP: geocoding, places, directions, and Street View.
- [memory-lancedb-mcp](https://github.com/cablate/memory-lancedb-mcp) — long-term memory for agents with hybrid retrieval.
- [Agentic-MCP-Skill](https://github.com/cablate/Agentic-MCP-Skill) — progressive MCP loading so agents can use tools without loading everything up front.

### Claude Code Ecosystem

- [claude-code-research](https://github.com/cablate/claude-code-research) — independent research on Claude Code internals and related tooling.
- [Claude-Code-Board](https://github.com/cablate/Claude-Code-Board) — a Kanban-style interface for operating Claude Code workflows.
- [ai-toolkit](https://github.com/cablate/ai-toolkit) — Claude Code skills and delivery patterns from real usage.
- [llm-atomic-wiki](https://github.com/cablate/llm-atomic-wiki) — a knowledge organization pattern built for LLM-assisted work.

### Applied AI Pipelines

- [banini-tracker](https://github.com/cablate/banini-tracker) — Facebook crawling, transcription, AI analysis, and multi-platform push for tracking public investment signals.
- [finance-viewer](https://github.com/cablate/finance-viewer) — a local-first finance workspace where AI does the first pass and humans make the final call.
