**CabLate** — 我打造 AI 基礎設施，開源它，然後用同一套東西經營自己的事業。<br>
I build AI infrastructure, open-source it, and run my business on it.

我做的不是單純的 AI 工具展示，而是把工具、工作流、內容、課程和交付系統串起來，讓 AI 從聊天介面變成能實際運作的工作環境。

**Proof:** 1,200+ public GitHub stars across MCP servers, Claude Code tooling, and applied AI pipeline projects.
